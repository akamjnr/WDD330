The Window Object - Chapter 9:

Browser Object Model: Collection of properties and methods that contain information about the browser and computer screen.

Global variables are variables created without using the const, let or var keywords and are actual properties of a global object. You should refer to global variables without using the window object (less typing and portable code between environments); except when it has not been defined.

Dialogs: 3 functions that produce dialogs in browsers and are supported in major browsers as methods of the window object: alert(), confirm() and prompt().

1. window.alert() method will pause the execution of the program and display a message (provided as an argument) in a dialog box.

2. window.confirm() method stops the execution of the program, displays a confirmation dialog with a message (provided as an argument), and options of OK or Cancel.

3. window.prompt() method stops the execution of the program, then displays a dialog with a message (provided as an argument), and an input field that allows the user to enter text.

Browser History: window.history property can be used to access information about previously visited pages in current browser session.

▪ window.history.length property shows the number of pages visited before the current page.

Controlling Windows:
▪ window.open() method opens a new window.
▪ close() method closes a window if you have a reference to it.
▪ window.moveTo() method moves a window, takes 2 parameters (X, Y screen coordinates).
▪ window.resizeTo() method resizes a window, takes 2 parameters (width, height).

Screen Information: object contains information about the screen the browser is displayed on. Screen object is more useful for mobile devices. It allows to turn off the device's screen, detect a change in orientation, lock it in a specific orientation.

Document Object: Each window object contains a document object.
document.write()
• write() method writes a string of text to the page.
• Use of document.write() is heavily frowned upon as there are extremely rare legitimate uses of it.

Cookies: Small text files that are saved/stored locally on a user's computer and can be retrieved between requests. Restricting cookies stops sites from accessing information about users, such as other sites they have visited.

Cookies can be used for personalizing user browsing experience, save user preferences, keep track of user choices, track users. “Cookies are session cookies by default”, meaning they are deleted once a browser session is finished (user closes browser tab or window). A cookies can only be read by the domain that created it.

Animation: window.requestAnimationFrame() method can be used to produce smooth and optimized animation by utilizing the browser's built-in graphics capabilities and can be cancelled using the cancelAnimationFrame() method.
